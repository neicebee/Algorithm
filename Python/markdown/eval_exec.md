# 💻 eval()과 exec()

- 파이썬 내장함수
- 문자열로 표현된 파이썬 코드 실행 시 사용하는 함수

## 👉🏻 eval()

- evaluate의 약자
  - 말 그대로 값을 구하는 함수
- 파라미터로 '식'을 받아 컴파일 코드로 변환함
  - exec()와 달리 '문'을 파라미터로 받을 수 없음
- 값을 반환하는 함수

    ```python
    a = 5
    b = 6
    a = eval('a+b')
    print(a)

    >>> 11
    ```

## 👉🏻 exec()

- execute의 약자
  - 말 그대로 실행하는 함수
- 파라미터로 받은 '문자열'을 컴파일 코드로 변환하여 실행함
- 값을 반환하지 않는 함수

    ```python
    a = 5
    b = 6
    exec("print(a, '+', b, '=', a+b);a+=1;b+=1;"*3)

    >>> 5 + 6 = 11
        6 + 7 = 13
        7 + 8 = 15
    ```
